Palanhar Dairy & Agricultural Farm Pvt. Ltd.

Website — Feature & Change Documentation

Repository: github.com/keshavvin/palanhar  |  Stack: React 19 + Vite + Tailwind CSS v4 + Framer Motion + Recharts  |  Document date: 12 June 2026

1. Overview

The website was rebuilt from a basic marketing site into an investor-pitch demo implementing the "Investor Management & Dividend Distribution System" SRS, plus a repaired and polished marketing shell, a brand loading experience, and a Gau Seva awareness page. All investor features are frontend demos driven by one mock-data file — no backend is required to present the site.

6. How to Run

  • Install once: npm install
  • Development: npm run dev → http://localhost:5173
  • Production build: npm run build → output in dist/ (deploy to any static host)

Demo notes: OTP accepts any 6 digits; approvals, transfers, downloads and exports are simulated in the browser. All money figures are illustrative. Dividends in the live product depend on company performance and Board approval.
